# FreshKeeper

Стартовый Android-проект (Kotlin + Jetpack Compose) для приложения контроля срока годности продуктов.

## Что уже реализовано
- Базовая архитектура: MVVM + Repository.
- Локальная база данных Room для хранения продуктов.
- Hilt для dependency injection.
- Экран добавления продукта: название, количество, срок (YYYY-MM-DD), опциональный штрихкод.
- Быстрые шаблоны добавления (молоко, яйца, сыр).
- Список продуктов по секциям: просроченные, скоро испортятся, свежие.
- Подключены зависимости для ML Kit barcode scanning и WorkManager.
- Базовый планировщик локальных уведомлений о сроке годности (за 3 дня, за 1 день и в день срока).

## Следующие шаги
1. Сканер штрих-кода через камеру.
2. Экран настроек уведомлений (время, частота, типы).
3. Улучшение аналитики по запасам и фильтры по категориям.


## Как проверить уведомления
1. На главном экране откройте блок **"Проверка уведомлений"**.
2. На Android 13+ нажмите **"Выдать разрешение на уведомления"**.
3. Нажмите **"Отправить тестовое уведомление"** — уведомление должно прийти примерно через 10 секунд.
