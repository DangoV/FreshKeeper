# FreshKeeper

Стартовый Android-проект (Kotlin + Jetpack Compose) для приложения контроля срока годности продуктов.

## Что уже реализовано
- Базовая архитектура: MVVM + Repository.
- Локальная база данных Room для хранения продуктов.
- Hilt для dependency injection.
- Экран добавления продукта: название, количество, срок (YYYY-MM-DD), опциональный штрихкод.
- Быстрые шаблоны добавления (молоко, яйца, сыр).
- Список продуктов по секциям: просроченные, скоро испортятся, свежие.
- Подключены зависимости для ML Kit barcode scanning и WorkManager.
- Планировщик локальных уведомлений о сроке годности (за 3 дня, за 1 день и в день срока).
- Запрос системного разрешения на уведомления при первом запуске (Android 13+).
- Запрос системного разрешения к камере при первом запуске.
- Сканирование штрих-кода через камеру (ML Kit + CameraX) с автоподстановкой в поле штрихкода.
- Индикатор активного сканирования и управление фонариком на экране сканера.
- Экран (блок) настроек уведомлений: включение/типы/час уведомлений с сохранением.
- Планировщик напоминаний использует настройки пользователя (включенность, типы, час).
- Фильтрация списка продуктов по названию с мгновенным подсчетом найденных позиций.
