# FreshKeeper

Стартовый Android-проект (Kotlin + Jetpack Compose) для приложения контроля срока годности продуктов.

## Что уже реализовано
- Базовая архитектура: MVVM + Repository.
- Локальная база данных Room для хранения продуктов.
- Hilt для dependency injection.
- Экран добавления продукта: название, количество, срок (YYYY-MM-DD), опциональный штрихкод.
- Быстрые шаблоны добавления (молоко, яйца, сыр).
- Список продуктов дома с отображением дней до окончания срока.
- Подключены зависимости для ML Kit barcode scanning и WorkManager (для следующего этапа).

## Следующие шаги
1. Экран добавления продукта (вручную + шаблоны).
2. Сканер штрих-кода через камеру.
3. Планировщик локальных уведомлений о сроке годности.
4. Экран "Скоро испортится" с сортировкой и фильтрами.
